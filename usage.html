kif command

What's this

kif is a general utility for shogi record. It can read csa, kif, ki2. It write record or board in these format and SVG and EPS. It also extract game information such as game length, player name, and legal moves. See command sample below.

Usage

General

kif is a command line tool to operate shogi record. accepted format is csa, kif, ki2, etc. short help is available by 
 kif -help
or 
 kif --help

Sample

Read record by this extension and output to csa format 
	kif filename.ki2 -onvert csa > convert.csa
Multiple files on linux 
	for f in *.kif; do ./kif $f -convert > $f.csa ; done
on windosx ??? 
	for f in *.kif do kif %f -convert > $f.csa
Create svg file only commented position 
	kif filename.kif -forc 1..-1 -svg
Get leagal move list and count of specified ply 
	kif filename.kif -for 25 -format "%N%M"
Get game inforamtion 
	kif filename.kif -info "%b-%w-%d"

Syntax

 kif filename [options]
if - is given for filename, stdin is used as target. see -input

Options

each option has to start with - or --. singular - stands for stdin for input file
-convert csa
convert record to one of csa, kif, ki2, pico, usi?, fsen? format.
-for 1..-1
target specifire for -format option. argmument shoud be one of -for n, -for n..m. 1 stands for the initial position befor any move is played. -1 stands for the last position before the last move is played.
-forc 1..-1
same to -for, but skip positoins which does not have comment.
-format %p%m
show board information in a given format.
-help
show help message
-info %d-%b(%B)%w(%W)%l
show game information in a ginve format.
-input auto
change file type recoginition to one of auto, csa, kif, ki2, pico, usi?, fsen?. basically kif recognize the file type from its extention. it has to be specified if - is given as input filename
-version
show version

Escape in format

character start from \ or % has special meaning.

general

\n
newline
\\
single \

Game information

followings are available in argument of -info and -format
%b
black player name
%w
white player name
%B
black elo rating
%W
white elo rating
%e
event name
%t
title
%l
game length
%d
date
%s
start at
%f
finish at
%o
opening

Board information

followings are available only in argument of -format
%p
current board ply
%m
last move (already played move)
%n
next move (to be played)
%M
legal moves (list of legal moves)
%N
legal moves num
%c
comment
%K
Kakinoki boad
%C
CSA board
%P
pico board
%O
OhTeX board
%S
SVG board
